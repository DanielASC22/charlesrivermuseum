# Museum of Industry Website Revamp

Made by: Daniel Olusheki, Ryan Nguyen, Brando Arevalo, Aajinkya Naik, and Emmy Zhou.

A modern website redesign for the Museum of Industry, built during **DeisHacks 2026** where it won **Best Non-Technical Hack**! 🏆

🔗 [Devpost Submission](https://devpost.com/software/askdjaskd)
Watch our pitch: [YouTube](https://www.youtube.com/watch?v=kRTc182lkwk)

## About

This project is a complete revamp of the Museum of Industry's website, featuring a fresh design with improved navigation, responsive layouts, and a modern user experience.

## Tech Stack

- **React** + **TypeScript**
- **Vite** for fast builds
- **Tailwind CSS** for styling
- **shadcn/ui** for UI components
- **Framer Motion** for animations
- **Lovable Cloud** for hosting & backend
